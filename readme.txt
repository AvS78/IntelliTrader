Data source

https://drive.google.com/drive/folders/1kGuepEwbtREI5dwich3N334j7mf2BgwJ?usp=sharing

The link contains the zipped files of all the raw tick data for EUR/USD from 2010 to 2019

Steps to run the codes

1. Download and unzip the tickwise data from the above link
2. Run Data Processing.py and conver the data to a consolidated dataframe
3. For any model, choose a sampling frequency and run the codes
